\section{Выводы}

В ходе выполнения данной работы на примере программ \texttt{lab2\_n} и \texttt{lab2\_eps}, вычисляющих значения заданной функции, были рассмотрены базовые принципы построения программ на языке C и обработки вещественных чисел:

\begin{enumerate}
  \item Объявление и использование переменных, а также указателей на переменные.
  \item Использование циклов \texttt{while} и ветвлений \texttt{if}.
  \item Организация ввода/вывода.
  \item Реализация пользовательской функции ввода
  \item Работа с функциями из встроенной библиотеки <math.h>
  \item Разработка функций.
  \item Реализация математических функций.
  \item Выделение компонентов программ в отдельные файлы.
\end{enumerate}
